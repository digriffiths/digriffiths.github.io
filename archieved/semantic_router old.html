Overview

I contributed to the development of the Semantic Router, an open-source text classification model that serves as a superfast decision-making layer for LLMs and agents. This model utilizes the power of semantic vector space to expedite decision-making processes, routing requests based on semantic meaning rather than relying on slower LLM generations. The project features a hybrid approach, combining both sparse and dense vectors for text classification. I engineered a bespoke TF-IDF encoder without depending on standard libraries like sklearn, ensuring the model remains lightweight. Additionally, I implemented unit tests to safeguard the code's functionality as the project evolves.
